使用指南

1. 單檔案轉換

選擇一個 Markdown 檔案，輸入標題和日期，點擊轉換即可生成 Word 文檔。

2. 批量處理

選擇多個檔案，系統會自動為每個檔案生成對應的 Word 文檔。

3. 模板選擇

支援多種預設模板：預設、學術論文、技術報告、簡潔風格。

4. 設置配置

在設置頁面可以自訂各種選項，如自動清理、壓縮輸出等。

技術支援

支援格式： Markdown (.md) → Microsoft Word (.docx)

清理規則： 自動移除所有 cite 標籤和相關引用

輸出目錄： 所有生成檔案儲存在 output/ 資料夾

系統需求： Windows 10+, 支援現代瀏覽器
