# Markdown to Word 轉換工具 - 快速開始

## 🚀 啟動方式

### 方式 1：雙擊啟動（推薦）
1. 雙擊 `start.bat` 檔案
2. 伺服器自動啟動，Web 介面自動打開

### 方式 2：手動啟動
1. 打開命令提示符（cmd）
2. 執行：`.\bin\web_server.exe`
3. 在瀏覽器中打開：`http://localhost:8888/web_gui.html`

## 📝 使用指南

### 單檔案轉換
1. 選擇 "轉換" 頁籤
2. 上傳 Markdown 檔案
3. 輸入標題（可選）
4. 選擇日期（默認為今天）
5. 點擊 "開始轉換" - 檔案自動下載

### 快速轉換
1. 選擇 "快速轉換" 頁籤
2. 上傳 Markdown 檔案
3. 點擊 "預覽處理結果" 查看效果
4. 點擊 "開始快速轉換" - 檔案自動下載

### 預覽功能
- 點擊 "預覽" 按鈕查看轉換效果
- 左側：原始 Markdown
- 右側：清理後的預覽結果

## ✨ 功能特性

✅ 自動清理 cite 標籤  
✅ 自動添加標題和日期  
✅ 自動下載轉換後的檔案  
✅ 實時預覽效果  
✅ 支援數學方程式  
✅ 轉換歷史記錄

## 📂 檔案位置

- 轉換後的檔案保存在：`output` 資料夾
- 根據日期和時間自動生成檔案名

## ⚙️ 系統要求

- Windows 10 或更高版本
- 現代瀏覽器（Chrome、Edge、Firefox 等）
- 8888 端口未被佔用

## 🔧 故障排除

### 頁面無法打開
1. 確認伺服器正在執行（應該有一個命令行視窗）
2. 檢查瀏覽器是否安裝了最新版本
3. 清除瀏覽器緩存（Ctrl+Shift+Delete）

### 轉換失敗
1. 確保 Markdown 檔案格式正確
2. 檢查 output 資料夾是否存在
3. 查看瀏覽器控制台（F12）中的錯誤信息

### 檔案無法下載
1. 檢查瀏覽器下載設置
2. 確認磁碟有足夠空間
3. 嘗試清除瀏覽器快取

## 📊 轉換統計

轉換過程中會顯示：
- 移除的 cite 標籤數量
- 原始字元數
- 清理後字元數
- 節省的空間

## 💡 提示

1. 使用 "快速轉換" 快速處理簡單文檔
2. 使用 "單檔案轉換" 自訂標題和日期
3. 在 "歷史記錄" 頁籤查看所有轉換
4. 在 "設置" 頁籤配置偏好選項
